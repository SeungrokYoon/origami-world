## Main Features

- User Authentication with Firebase
  - Sign up
  - Sign in
  - Sign out
  - Reset password
- Create, Read, Update, Delete origami instructions
- Infinity Scroll
- Origami instruction editor
- Instruction Slide show UI
- Like button for each instruction post
- Leader board sorted by 'likes'
